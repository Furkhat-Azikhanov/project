# Web-приложение для определения заполненных форм

Это веб-приложение на Flask предназначено для определения подходящих шаблонов форм на основе входных данных, передаваемых через POST-запрос. Если подходящий шаблон не найден, приложение автоматически определяет типы полей на основе правил валидации.

## Функционал

1. **Добавление шаблонов форм**: Вы можете добавлять новые шаблоны форм с полями и их типами через API.
2. **Поиск подходящего шаблона**: Приложение ищет шаблон, который полностью соответствует входным данным (имена полей и их типы).
3. **Автоматическая типизация полей**: Если шаблон не найден, приложение возвращает типы переданных данных.
